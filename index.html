Celestia Namespace Format

Celestia namespaces must follow a specific format:

  • Total length: 29 bytes (1 byte version + 28 bytes ID)
  • Version 0: Must have 18 leading zero bytes in the ID
  • Reserved namespaces cannot be used

The options below will automatically create valid namespaces for you.
